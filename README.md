# caezium/homebrew-tap

Homebrew tap for [Burrow](https://github.com/caezium/Burrow) — a free,
open-source macOS GUI for the Mole (`mo`) CLI.

```bash
brew install mole
brew install --cask caezium/tap/burrow
```

Burrow is unsigned (pre-1.0); the cask clears the Gatekeeper quarantine
flag on install. See the [main repo](https://github.com/caezium/Burrow) and
its [SECURITY.md](https://github.com/caezium/Burrow/blob/main/SECURITY.md).
