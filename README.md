# caezium/homebrew-tap

Homebrew tap for caezium's macOS apps.

## [Wisp](https://github.com/caezium/wisp)

On-device inline AI autocomplete for any macOS text field.

```bash
brew install --cask caezium/tap/wisp
```

## [Burrow](https://github.com/caezium/Burrow)

A free, open-source macOS GUI for the Mole (`mo`) CLI.

```bash
brew install mole
brew install --cask caezium/tap/burrow
```

Both apps are unsigned (pre-1.0); the casks clear the Gatekeeper quarantine
flag on install.
